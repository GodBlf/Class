# chrome
| 快捷键            | 功能说明                                 |
|-------------------|------------------------------------------|
| **Ctrl + T**       | 打开新标签页                            |
| **Ctrl + W**       | 关闭当前标签页                          |
| **Ctrl + Shift + T** | 恢复关闭的标签页                      |
| **Ctrl + Tab**     | 切换到下一个标签页                      |
| **Ctrl + Shift + Tab** | 切换到上一个标签页                  |
| **Ctrl + L**       | 快速选中地址栏内容                      |
| **Ctrl + D**       | 收藏当前网页                            |
| **Ctrl + F5**      | 强制刷新，不使用缓存                    |
