# 编译器选项

1. 开启 C++11 或更高版本标准
这是最直接的解决办法，适用于绝大多数版本的 Dev-C++。

点击菜单栏的 工具 (Tools) -> 编译选项 (Compiler Options)。

在弹出的窗口中，选择 代码生成/优化 (Settings) 选项卡。

在下方的子选项卡中点击 代码生成 (Code Generation)。

找到 语言标准 (-std) (Language standard) 这一行。

在右侧的下拉菜单中选择 GNU C++11，或者更高版本

点击 确定 (OK)，然后重新编译你的代码。

2. release 和 debug都需要改 release就改个版本就行
3. 优化级别:debug
4. 链接器产生调试信息
